## 16. Content Patterns

### Section Title Pattern
```
Title: Short, impactful (3-6 words)
Subtitle: Explanatory, benefit-focused (8-15 words)
```

### Card Content Pattern
```
Title: Product/Feature name
Tagline: One-line value proposition (italicized)
Description: 1-2 sentences expanding on benefits
CTA: Action verb + object
```

### CTA Band Pattern
```
Title: Question or invitation (addressing user need)
Subtitle: Value proposition or benefit statement
Primary CTA: Action-oriented button
Secondary CTA: Alternative action (optional)
```

### Stat Display Pattern
```
Large number/percentage (bold, colored)
Short label (1-3 words)
Optional: Small description
```
